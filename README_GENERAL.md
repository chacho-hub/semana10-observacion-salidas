# Semana 10 - Observación de resultados y salidas

Este paquete contiene un proyecto web para trabajar la sección de la Unidad 2:

**Observación de resultados y salidas; interpretación de mensajes; validación de objetivos.**

## Estructura

```text
semana10-observacion-salidas/
├── carpeta_estudiante/      # Material para entregar al estudiante
└── carpeta_profesor/        # Guía detallada, pasos de solución y versión final
```

## Uso sugerido

- Entrega a los estudiantes únicamente la carpeta `carpeta_estudiante/`.
- Conserva `carpeta_profesor/`, porque contiene la guía paso a paso, las respuestas del micro-quiz, los cambios exactos y la solución final.

## Publicación en GitHub

El archivo `.gitignore` ya excluye `carpeta_profesor/` para que puedas publicar solamente el material del estudiante.
